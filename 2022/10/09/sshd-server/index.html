install openssh-server for linux error

install openssh-server in linux error

Error is displayed:

cause

The sshd_config in /etc/ssh is not exist

Workaround

  1. Copy sshd_config in other linux device
  2. write sshd_config in /etc/ssh use vim
  3. /usr/sbin/sshd

if error is displayed when you run 4.
sshd: no hostkeys available -- exiting.

Assign a public key for your device
ssh-keygen -A

At last
service ssh start
